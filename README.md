Objetivo: Sistema web pra registrar e acompanhar dados dos alunos (fotos, ocorrencias, interacoes, etc).

SPRINT 1 - 02 e 04/09

Back-End: Configuracao da API (index, server, routes).

LIMA: Prototipo no Figma + pagina de login (HTML/CSS).

SPRINT 2 - 07 e 09/10
