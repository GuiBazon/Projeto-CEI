O Projeto CEI (Carômetro Escolar Interativo) foi desenvolvido para a aprendizagem e treinamento pelo SENAI.

O objetivo principal era fazer um sistema capás de registrar Usuarios (docentes), Turmas, Alunos e Ocorrencias.
Mas devido ao tempo curto o projeto foi reduzido a uma API completa de acordo com os conhecimentos adquiridos no SENAI,
e um Front não funcional do prototipo inteiro, exceto pelo login e o cadastro.
